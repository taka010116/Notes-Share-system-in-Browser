ブラウザで完結する、簡易的な演奏ゲームを作りたい。
BMSのように、ノーツとwavファイルを紐づけることで演奏間を実装します。
まだまだ開発途中ですが、大まかな基盤を作成しました。

曲データ
このプログラムに対応する曲フォルダは、jsonファイル、wavファイル、bmpファイルで構成されています。

エディター
![Editor](editor.png)

こちらは、譜面作成画面になります。(対応：editor.html)
曲データを読み取り、そこにあるwavファイル一覧をWAVで表示しています。
各ノーツにwavファイルを紐づけていきます。
左から5個のレーンは、プレイヤーが演奏するノーツになります。プレイヤーがキーをたたいた時、対応する音を鳴らします。
そこから、右4つのレーンは、バックで流す音源になります。プレイヤーの意思にかかわらず、指定したタイミングで音を鳴らします。
最後に、一番右のレーンは、bmpファイルを設定するものになります。
bmpは、画像データであり、パラパラ漫画の要領でミュージックビデオ(通称：BGA)を作成することができます。

作成した譜面データは、jsonファイルで保存され、それを曲フォルダに入れることで完成となります。
